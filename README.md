# 🎬 Disney+ Clone (Landing Page)

A responsive **Disney+ Clone Landing Page** built to replicate the look and feel of the official Disney+ homepage.  
This project focuses on UI design, layout structure, and responsiveness using modern web development practices.

---

## 🚀 Project Status
🟡 **In Progress**  
✔️ Main landing page completed  
❌ Authentication, video streaming, and backend features are yet to be implemented

---

## ✨ Features Implemented
- 🎨 Disney+ inspired UI
- 📱 Fully responsive design (mobile, tablet & desktop)
- 🖼️ Hero banner with call-to-action
- 🎞️ Platform brand sections (Disney, Pixar, Marvel, Star Wars, National Geographic)
- 💡 Clean layout using reusable components
